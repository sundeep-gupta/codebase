SOFTWARE REQUIREMENTS

1. Perl version 5.8.8 and above needs to be Installed and Perl.exe should be present in the PATH variable.

2. The below mentioned modules which are not part of the standard perl distribution also need to be installed.

3. Outlook 2003 should be installed and activated.(For the Clean and Spam Email Test)

4. The system should have an administrator and UAC disabled on Vista. Also a limited user with the user name "user" and password "a" must be created before starting the tests.

5. The system name should be changed in case two or more people are using the same image.

6. Autologon should be enabled with the administrator as the default user. Use the command "control userpasswords2".

7. Adobe acrobat reader should be installed.

 

Perl Modules to be Installed:

  1. SpreadSheet::WriteExcel
  2. Getopt::Long;
  3. Win32::AbsPath
  4. Time::HiRes
  5. Config::INI::Simple
  6. Config::Tiny
  7. Win32::IEautomation
  8. Time::Format
  9. Win32::Process::List
