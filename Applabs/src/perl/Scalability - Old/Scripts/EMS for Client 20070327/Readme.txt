Instruction to install Apache & EMS on the client

1. Unzip http_root.zip in c:\orbital, all files go into c:\orbital\http_root
2. Unzip Apache.zip in c:\, all files go into c:\Apache
3. cd c:\Apache
4. Apache -i Apache
5. net start Apache
6. Enter on the browser URL http://<clientip>:8085/index_debug.php
