Variables

Configuration variables
Configuration variables are in the file named "cnfg_vars.php" .
This file is for variables that hold info like DB connection info, url paths, and settings to help customize the look of the site.

Global variables
I wanted to try to keep the usage of global symbols down to a minimum.
So what I did was create a global hash, or what you might call an associative array, to hold most of the global variables that the application uses. This array is named "$gbl". It is located in the "lib" directory in the file named "vars_gbl.php" .
Every once in while I assign values to elements that don't already exists in the $gbl array, or in other words are not predefined. This is just a convenient place to store values in certain situations. So for instance I might do something like this:
$gbl['misc']['trueOrFalse'] = true;
