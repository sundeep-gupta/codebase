Security
It is recommended that the "lib", "cnfg", "admin_scripts", and "templates" directories be put above your web root if possible. At the very least "cnfg" and "admin_scripts" should be password protected.
Also the "admin" directory should stay where it is, but must be password protected. If you do not password protect this directory anyone who knows the url will be able to delete ads from your database! That's definitely not a good thing.
