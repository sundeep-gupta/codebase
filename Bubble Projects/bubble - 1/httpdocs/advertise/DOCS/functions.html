Functions

Some functions are only needed in one file. But the functions that are needed in more than one file will either be in a file named "func_descriptiveName.php" or in "func_common.php" . If you want to write a function that should be required in all or most of the other files, then you should put it in "func_common.php", since "func_common.php" is already "required" in almost every file of the application.
