Daily Summary Report
This report displays
  • Number of checks,
  • Success count,
  • Failure count,
  • Availability and
  • Average response time in milliseconds
at each location, on current date, for the service selected.

Table of records
The Table of records consists of following columns
  • TimeStamp - time at which the response is obtained.
  • DNStime - time taken for the url to resolve into IP address.
  • ConnectTime - taken to connect to server.
  • FirstByte - time taken to get the first byte of response from server.
  • DownloadTime - time taken to get the whole bytes of response.
  • ResponseTime - time taken to get the response from server.
  • Status - True or false; success report of the URL.
  • ResponseCode - Http response code_Http response message.
